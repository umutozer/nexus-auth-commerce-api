# Kalite Kontrol Listesi
## HTML Validasyonu
- [ ] Tüm HTML etiketleri đúng syntax ile yazılmıştır.
- [ ] Tüm HTML etiketleri kapatılmıştır.
## CSS Uyumluluğu
- [ ] Tüm CSS stilleri tutarlı ve doğru bir şekilde uygulanmıştır.
- [ ] Tüm CSS stilleri farklı tarayıcılar ve cihazlarda doğru çalışmaktadır.
## JS Hataları
- [ ] Tüm JavaScript kodları hata olmadan çalışmaktadır.
- [ ] Tüm JavaScript fonksiyonları beklenen sonuçları vermektedir.
## Responsive Tasarım
- [ ] Tüm sayfalar mobil, tablet ve masaüstü cihazlarda doğru görünmektedir.
- [ ] Tüm sayfalar farklı ekran boyutlarında doğru bir şekilde scale olmaktadır.
## SEO
- [ ] Tüm sayfaların meta tagları (title, description, keywords) doğru bir şekilde doldurulmuştur.
- [ ] Tüm sayfaların heading hierarchy (h1, h2, h3, vb.) doğru bir şekilde uygulanmıştır.
- [ ] Tüm resimlerin alt textleri doldurulmuştur.
## Güvenlik
- [ ] Sistem XSS saldırılarına karşı güvenlidir.
- [ ] Sistem CSRF saldırılarına karşı güvenlidir.
## Performans
- [ ] Sayfaların yükleme hızı 3 saniyeyi geçmemektedir.
- [ ] Tüm resimlerin boyutları optimize edilmiştir.
- [ ] Tüm CSS ve JavaScript dosyaları minify edilmiştir.
## Erişilebilirlik
- [ ] Sistem WCAG 2.1 AA standardına uygunluk göstermektedir.
- [ ] Tüm öğeler için ARIA atributları doğru bir şekilde kullanılmıştır.
